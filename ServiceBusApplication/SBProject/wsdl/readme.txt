Este archivo se ha colocado aquí como parte de la creación de una nueva carpeta. 
Debido a que las carpetas vacías en los proyectos se ocultan por defecto, este archivo 
permite que la carpeta sea visible inicialmente. Una vez que la carpeta tenga
archivos adicionales, este readme.txt ya no será necesario y se podrá suprimir.

Puede impedir la creación del archivo readme.txt en el futuro 
configurando la ventana de aplicaciones para que muestren las carpetas vacías. 
Para ello, active el valor "Mostrar Carpetas Vacías" en las 
opciones de visualización del panel Proyectos.
